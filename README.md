# Hueku

> **Hueku** is a minimalist color palette and combination explorer inspired by the work of **Sanzo Wada** (1883–1967). 

This application offers a modern, tactile, and highly responsive interface to explore Wada's collection of traditional Japanese color palettes and combinations from his seminal 1935 publication, *A Dictionary of Color Combinations*.

---

## ✦ Key Features

- **Minimalist Design System**: Built with a clean, grid-based aesthetic that puts the focus entirely on the colors.
- **Color Detail Inspector**: View precise values for each color across multiple color spaces including **HEX**, **RGB**, **HSL**, and **CIELAB**.
- **Dynamic Combination Explorer**: Discover Sanzo Wada's curated 2-color, 3-color, and 4-color combinations that contain any selected color.
- **Responsive Layout**: Designed to adapt fluidly from mobile viewports to ultra-wide desktop monitors with zero visual clutter.
- **Modern Typography**: Styled using the elegant *Jost* typeface to match the design's typographic weight system.

---

## ✦ Tech Stack

- **Core**: [Vue 3](https://vuejs.org/) (Composition API with `<script setup>`)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Routing**: [Vue Router](https://router.vuejs.org/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Styling**: Vanilla CSS (custom design tokens and utility grid)

## ✦ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

Clone the repository and install the dependencies:

```bash
# Install package dependencies
npm install
```

### Development

Run the local development server:

```bash
# Start Vite development server
npm run dev
```

### Production Build

Build the production bundle for deployment:

```bash
# Compile and minify for production
npm run build
```
